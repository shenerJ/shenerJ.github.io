hexo配置踩坑帖

执行 hexo new '标题' 创建文章之后，如何快捷打开进行编辑呢？

解决步骤：

  1. 在 Hexo 目录下的 scripts目录（ 没有，就新建一个，mkdir scripts ）中创建 js 脚本文件, js 文件名任意，如 auto_open_editor.js

  2. auto_open_editor.js 内容：

    Ps.

    1. Hexo 版本使用 hexo version 查询
    2. 编辑器绝对路径获取方法：（Mac）直接将 应用程序 中的编辑器拖进终端，即可获取，如 Typora 的路径为：/Applications/Typora.app

配置完毕后，创建文章即可自动打开编辑器了。
